GTA V User Music Manager

What this app does
- Helps you manage GTA V / GTAV Enhanced User Music tracks in one place.
- Lets you search and download songs from YouTube.
- Renames downloads into a clean "Artist - Song" format when possible.
- Lets you preview, search, sort, and delete tracks from your User Music folder.

Supported download modes
1. Single song search
   Type a song name. For better results, include the artist name too.
   Example: Luis Fonsi Despacito

2. YouTube video URL
   Paste a direct YouTube song/video URL.

3. YouTube playlist URL
   Paste one playlist URL to download the full playlist.
   Only one playlist should be used at a time.

4. Batch download
   Type multiple entries separated by commas.
   Example: Luis Fonsi Despacito, The Weeknd Blinding Lights, Adele Hello

How to use
1. Run "GTA Muisc Manager.exe"
2. Check the folder path at the top.
3. If the default GTA User Music folder was not found, click "CHANGE FOLDER" and set it manually.
4. Use the big input box to search songs or paste URLs.
5. Click "SYNC LIBRARY" to download and import tracks.
6. Use the Stored Tracks search box to filter your library.
7. Use the sort dropdowns to sort by date added, alphabetical order, or song length in ascending or descending order.
8. Use the play button to preview a track.
9. Use the red hover X button to delete a selected song file from the currently shown library list.

Notes
- The app keeps its config file in the same folder as the exe.
- Keep ffmpeg.exe, ffplay.exe, and ffprobe.exe in the app folder.
- Internet is required for downloads.
- Search matching in the library is typo-tolerant for small spelling mistakes.
- If a downloaded filename already exists, the app will keep both files by creating a numbered name instead of overwriting the old one.

Safety
- Deletes only target the exact song file selected in the visible list.
- Even so, always keep backups of your music if it matters to you.
